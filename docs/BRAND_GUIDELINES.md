# Tend Brand Guidelines (Current Direction)

## Name
- Product name: `Tend`

## Color system
- White (background): `#FFFFFF`
- Grow green (primary): `#4CAF7D`
- Morning gold (accent): `#F0C060`
- Surface (cards): `#F5F5F3`
- Near black (text): `#1A1A1A`
- Dark background: `#0F0F0F`
- Muted secondary text: `#888884`

## Typography
- Display: Plus Jakarta Sans 700
- Heading: Plus Jakarta Sans 500
- Body: Inter 400
- Caption: Inter 400

## Tone
- Grounded
- Clear
- Hopeful
- Practical

## Current implementation notes
- Design tokens are defined in:
  - `WalkWorthy/DesignSystem/WWColor.swift`
  - `WalkWorthy/DesignSystem/WWTypography.swift`
- Font files should be dropped in:
  - `design/brand-assets/fonts/`
  - then synced via `./scripts/sync_brand_assets.sh`
